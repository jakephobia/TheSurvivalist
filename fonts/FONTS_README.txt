FONT FILES REQUIRED
==================

Download these fonts (free for personal use) and place the .ttf files in this folder:

1. BIOTRIP (for body text)
   https://www.dafont.com/it/biotrip.font
   Required: Biotrip-Regular.ttf, Biotrip-Bold.ttf

2. CHOLERIC (for titles)
   https://www.dafont.com/it/choleric.font
   Required: Choleric-Regular.ttf

Ensure the file names match exactly for the CSS to load them.
